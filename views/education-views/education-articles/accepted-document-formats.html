Educational Coordinator Guide

Accepted Documentation Formats

Set expectations for students and faculty so every upload matches CPNW’s minimum standards and can be cleared by DVS reviewers or in-house coordinators the first time.

Core information every document needs

Whether you review requirements yourself or rely on a Document Verification Specialist, each record must stand on its own with the following data points visible:

  • Student’s full legal name matching their CPNW profile.
  • Healthcare provider or lab name, plus contact information.
  • Exam, vaccine, or lab title (for example, “MMR Titer”).
  • Date of service; multi-dose series should show each dose.
  • Result or provider signature confirming completion.

Approved file types

Ask students to combine multi-page records into one upload using these formats so the Review table stays organized:

  • .pdf
  • .docx
  • .txt
  • .pptx
  • .xlsx
  • .jpg
  • .jpeg
  • .png

These formats preserve readability when downloaded or moved into a student’s Docs folder for future audits.

Submissions that are rejected

To keep the Clinical Passport defensible with healthcare partners, reviewers decline records that:

  • Are handwritten or contain edits/white-out that skew the original content.
  • Use screenshots from portals that do not show the student name or service date.
  • Will expire before the clinical start date or omit dose details.
  • Contain multiple students’ records in the same file.

Help students submit it right the first time

  1. Share this checklist with every cohort during onboarding so everyone knows the standards before starting uploads.
  2. Encourage students to preview their file inside CPNW after uploading; if text looks blurry to them, it will to reviewers as well.
  3. Remind them that every rejection includes guidance. Have them read the note carefully, make the fix, and resubmit in a single file so the audit trail stays clear.
Tip: If your program purchased DVS services, the specialists enforce the same standards. Coordinators still see every decision, making it easy to verify students once their CPNW requirements are approved.
